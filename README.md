# Verify and Celebrate

Интерактивный квиз для верификации личности с забавными вопросами и изображениями.

## Технологии

Проект построен с использованием современного стека технологий:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Локальная разработка

Для запуска проекта локально:

```sh
# Установите зависимости
npm install

# Запустите сервер разработки
npm run dev
```

## Деплой

Проект автоматически деплоится на GitHub Pages при пуше в ветку main.
